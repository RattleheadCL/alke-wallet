# 🚀 Alke Wallet

Una billetera digital moderna desarrollada como proyecto integrador, diseñada para gestionar saldos, depósitos y transferencias de forma simple, rápida y segura.

---

## 📖 Descripción

**Alke Wallet** es una aplicación web que permite a los usuarios administrar sus fondos digitales mediante una interfaz intuitiva y amigable.

El proyecto fue desarrollado aplicando buenas prácticas de desarrollo frontend y gestión de código con Git y GitHub, simulando el funcionamiento básico de una billetera electrónica.

---

## ✨ Características Principales

### 🔐 Inicio de Sesión

* Acceso seguro mediante autenticación de usuarios.
* Validación de credenciales.
* Gestión de sesión.

### 💰 Gestión de Saldo

* Visualización del saldo disponible.
* Actualización dinámica de fondos.

### 📥 Depósitos

* Registro de depósitos realizados por el usuario.
* Actualización automática del balance.

### 📤 Transferencias

* Envío de fondos entre cuentas.
* Validaciones de saldo disponible.
* Registro de movimientos realizados.

### 📊 Historial de Transacciones

* Consulta de movimientos recientes.
* Seguimiento de depósitos y transferencias.

### 📱 Diseño Responsivo

* Compatible con dispositivos móviles, tablets y computadores.
* Interfaz desarrollada con Bootstrap 5.

## 🌱 Flujo de Trabajo Git

El proyecto utiliza una estrategia basada en ramas para mantener un desarrollo organizado.

### Ramas principales

| Rama                  | Descripción                             |
| --------------------- | --------------------------------------- |
| main                  | Versión estable del proyecto            |
| feature/login         | Desarrollo del sistema de autenticación |
| feature/transacciones | Envío y recepción de fondos             |
| feature/depositos     | Gestión de depósitos y saldo            |

## 🚀 Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/RattleheadCL/alke-wallet.git
```

2. Entrar al directorio:

```bash
cd alke-wallet
```

3. Abrir el archivo principal:

```bash
index.html
```

o utilizar Live Server en Visual Studio Code.
